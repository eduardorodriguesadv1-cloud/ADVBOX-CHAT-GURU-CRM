Quero expandir a integração com Meta Marketing API pra puxar dados completos de cada campanha. Hoje eu já tenho campaigns + métricas básicas. Preciso adicionar mais detalhes.

NOVAS TABELAS:

1. ad_sets (conjuntos de anúncios)
- id (serial PK)
- meta_adset_id (text, unique)
- campaign_id (FK pra campaigns)
- name (text)
- status (text)
- daily_budget (decimal)
- targeting_summary (text) — resumo da segmentação
- audience_type (text) — "open" / "saved" / "custom" / "lookalike"
- audience_details (jsonb) — dados completos da audiência
- age_min, age_max (integer)
- genders (text) — "all" / "male" / "female"
- locations (jsonb) — localizações
- created_at, updated_at

2. ads (anúncios individuais)
- id (serial PK)
- meta_ad_id (text, unique)
- adset_id (FK pra ad_sets)
- name (text)
- status (text)
- creative_id (text)
- creative_type (text) — "image" / "video" / "carousel"
- creative_url (text) — URL do criativo
- headline (text)
- body (text) — texto do anúncio
- call_to_action (text)
- created_at, updated_at

3. ad_metrics_daily (métricas avançadas por anúncio)
- id (serial PK)
- ad_id (FK pra ads)
- date (date)
- spend (decimal)
- impressions (integer)
- reach (integer)
- clicks (integer)
- ctr (decimal) — click-through rate
- cpc (decimal) — cost per click
- cpm (decimal) — cost per mille (mil impressões)
- frequency (decimal)
- conversations (integer)
- created_at
